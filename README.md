## 📄 网页自动刷新（高级增强版）

> 每隔一段时间自动刷新当前网页，支持自定义刷新间隔，内置倒计时显示、暂停/重置按钮，并通过油猴菜单控制启用与设置。适用于挂机、心跳保持、PT站等使用场景，安全稳定、简洁高效。

---

### 🚀 功能特点

* ✅ **新增右下角控制面板“⚙ 设置”按钮**，可直接修改当前页面刷新间隔(v1.1)
* 🧠 每个网址可单独设置刷新时间，互不影响，支持多页面独立配置(v1.1)
* ✅ 支持用户自定义刷新间隔时间（通过油猴菜单设置）
* ✅ 页面右下角显示倒计时剩余时间
* ✅ 可手动暂停 / 继续刷新
* ✅ 可一键重置刷新时间
* ✅ 支持浏览器标题栏倒计时提醒
* ✅ 所有设置持久保存，不用每次刷新都重新配置
* ✅ 自动判断刷新时间是否合理，避免死循环
* ✅ **默认不开启刷新**，仅在你主动设置后才运行，非强制、不打扰

---

### 🧩 使用方法

#### 第一步：安装脚本

1. 点击页面上的【安装此脚本】按钮
2. 在你想启用自动刷新的网页打开后，点击浏览器右上角的油猴图标，找到本脚本名称

#### 第二步：设置刷新时间

* 点击脚本名称下的「📥 设置刷新间隔时间（秒）」
* 输入你想要的刷新间隔，例如 `60`（表示每隔60秒刷新一次）
* 保存后刷新页面，自动启用

#### 第三步：可选功能菜单

* `关闭自动刷新（设为0）`：停用该功能，刷新页面即生效

---

### 💻 页面UI控制面板

在启用自动刷新后，网页右下角会出现一个小面板，包含以下内容：

* ⏱ 倒计时：显示剩余时间
* ⏸ 暂停：可暂停或恢复刷新
* 🔁 重置：将倒计时重置为初始设置值

---

### 📌 注意事项

* ✅ 最小刷新时间为 **5秒**，防止误设成 `0` 导致死循环
* ✅ 不会干扰页面任何功能，代码轻量不影响性能
* ✅ 默认不生效，需你主动设置后才启用
